About Whats-On-Netflix

Whats-On-Netflix is a data analytics project designed to provide insightful visualizations and trends from Netflix's extensive content library worldwide. Our goal is to help viewers explore streaming trends—covering movies and TV shows, their ratings, durations, and country distributions—in an engaging and interactive way.

Project Highlights

  • Cleaned and analyzed Netflix titles data for meaningful insights.
  • Visualized content types, release trends, duration distributions, and ratings.
  • Implemented an animated, Netflix-inspired web interface for smooth user experience.
  • Built with Python, Flask, Pandas, Matplotlib, HTML, CSS, and JavaScript.
